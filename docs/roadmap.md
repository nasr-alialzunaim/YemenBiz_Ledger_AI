# خارطة الطريق

## المرحلة 1: MVP

الموجود حاليًا:

- Backend FastAPI.
- SQLite.
- إدارة العملاء.
- إدارة المنتجات.
- إدارة المبيعات.
- إدخال بيع باللغة العربية.
- Dashboard.
- تقرير عربي.
- Streamlit Frontend.
- اختبارات أولية.

---

## المرحلة 2: تحسينات مهمة

- إصلاح وتحسين محلل اللغة العربية.

- دعم أكثر من عملة.
- إضافة الموردين.
- إضافة المصروفات.
- إضافة سند قبض.
- إضافة سند صرف.
- إضافة فواتير PDF.

## المرحلة 4: تطبيق جوال

- تطبيق Flutter للكاشير.

---

## المرحلة 5: ذكاء اصطناعي أعمق

- سؤال البيانات:
  - من أكثر عميل عليه دين؟
  - ما أكثر منتج مبيعًا؟
  - هل أبيع منتجًا بخسارة؟
- اقتراح أسعار.
- تنبيه عند انخفاض هامش الربح.
- توليد رسائل مطالبة بالدفع.
- تحليل اتجاهات المبيعات.
